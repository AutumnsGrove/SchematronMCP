ðŸŽ¯ QUICK START (5 Minutes)
--------------------------

1. EXTRACT THIS ZIP FILE
   - Extract to somewhere like ~/schematron-mcp
   - Open Terminal and cd to that directory

2. INSTALL DEPENDENCIES
   
   cd ~/schematron-mcp
   python3 -m venv venv
   source venv/bin/activate
   pip install -e .

3. TEST THE INSTALLATION
   
   python test_extraction.py
   
   (This will download the model ~2GB on first run)

4. CONFIGURE CLAUDE DESKTOP
   
   - Get your full path: pwd
   - Edit ~/.config/claude/claude_desktop_config.json
   - Add (replace YOUR_PATH):

   {
     "mcpServers": {
       "schematron": {
         "command": "python",
         "args": ["YOUR_PATH/schematron-mcp/server.py"],
         "env": {
           "SCHEMATRON_MODEL_PATH": "mlx-community/Schematron-3B-4bit"
         }
       }
     }
   }

5. RESTART CLAUDE DESKTOP

6. TEST IN CLAUDE
   
   Try: "Extract product data from this HTML: 
   <div><h1>iPhone</h1><p>$999</p></div>"

ðŸ†˜ TROUBLESHOOTING
------------------

Problem: "mlx_lm not found"
Solution: Make sure venv is activated: source venv/bin/activate

Problem: "Model download fails"
Solution: Check internet connection, try: 
  python -c "import mlx_lm; mlx_lm.load('mlx-community/Schematron-3B-4bit')"

Problem: Claude doesn't see the tool
Solution: 
  1. Check logs: ~/Library/Logs/Claude/mcp*.log
  2. Verify absolute path in config (not ~/)
  3. Restart Claude Desktop completely

Problem: Extraction fails
Solution: 
  1. Make sure HTML is valid
  2. Try with auto_clean=True
  3. Check schema is valid JSON Schema
